## 2. 服务清单（Table 3-1）

| 序号 | 服务名称 | 服务标识 | 端口 | 所属模块 | 主要职责 | 技术栈 |
|:---:|:--------|:--------|:----:|:--------|:--------|:-------|
| 1 | API网关 | atbs-gateway | 8080 | atbs-gateway | 路由转发、负载均衡、限流熔断、统一认证 | Spring Cloud Gateway |
| 2 | 认证服务 | auth-service | 8081 | auth-service | 用户认证、OAuth2授权、权限管理、JWT签发 | Spring Boot + Spring Security |
| 3 | 用户服务 | user-service | 8082 | user-service | 用户管理、乘客管理、员工管理 | Spring Boot + MyBatis-Plus |
| 4 | 订单服务 | order-service | 8083 | order-service | 订单管理、票务管理、支付回调 | Spring Boot + MyBatis-Plus |
| 5 | 航班服务 | airplane-service | 8084 | airplane-service | 航班管理、航线管理、座位预订 | Spring Boot + MyBatis-Plus |
| 6 | 预测服务 | prediction-service | 8000 | prediction-service | 客流预测、ARIMA模型训练 | FastAPI + Python |
| 7 | 工具模块 | atbs-tools | - | atbs-tools | 公共工具类、配置、拦截器 | Spring Boot |

## 4. 服务间调用关系

| 调用方 | 被调用方 | 调用方式 | 用途 |
|:------|:--------|:--------|:-----|
| Gateway | 所有服务 | 路由转发 | 统一入口 |
| Order Service | User Service | Feign Client | 获取用户信息 |
| Order Service | Airplane Service | Feign Client | 获取航班信息 |
| Airplane Service | Order Service | Feign Client | 查询订单状态 |
| User Service | Auth Service | Feign Client | 用户认证同步 |
| 所有服务 | Nacos | 服务注册/发现 | 服务治理 |

## 5. 模块结构说明

每个微服务采用分层模块结构：

```
service-name/                           # 服务根目录
├── service-name-api/                   # API模块（对外暴露）
│   └── src/main/java/
│       └── api/v1/                     # API版本控制
│           ├── app/                    # C端接口
│           ├── mgmt/                   # 管理端接口
│           ├── internal/               # 内部服务间调用
│           └── notify/                 # 回调通知接口
├── service-name-biz/                   # 业务逻辑模块
│   └── src/main/java/
│       ├── service/                    # 业务服务接口
│       ├── service/impl/               # 业务服务实现
│       ├── mapper/                     # 数据访问层
│       ├── client/                     # Feign客户端
│       └── state/                      # 状态机
├── service-name-ao/                    # 应用对象模块
│   └── src/main/java/
│       ├── req/                        # 请求对象
│       └── res/                        # 响应对象
├── service-name-bo/                    # 业务对象模块
│   └── src/main/java/
│       ├── eo/                         # 实体对象（Entity）
│       └── dto/                        # 数据传输对象
├── service-name-common/                # 公共模块
│   └── src/main/java/
│       ├── enums/                      # 枚举定义
│       └── constants/                  # 常量定义
└── service-name-boot/                  # 启动模块
    └── src/main/java/
        └── Application.java            # SpringBoot启动类
```

## 8. 命名规范

- **服务名**: `xxx-service`（如 auth-service, user-service）
- **模块名**: `service-name-xxx`（如 auth-api, auth-biz）
- **包名**: `top.flowerstardream.atbs.xxx`
- **API路径**: `/api/{client}/v1/{resource}`
  - `client`: app(客户端)/mgmt(管理端)/internal(内部)
  - `v1`: API版本
  - `resource`: 资源名称
