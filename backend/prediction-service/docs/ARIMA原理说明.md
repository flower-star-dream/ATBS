# ARIMA模型原理说明

## 决策依据说明

本文档阐述ARIMA模型的数学原理，为模型选型提供理论依据。

**核心决策问题**：为什么ARIMA适合航空客流预测？

**理论依据**：
1. 航空客流数据具有明显的时间序列特征（趋势性、季节性）
2. ARIMA专为时间序列设计，能够有效捕捉这些特征
3. ACF/PACF分析显示数据适合ARIMA建模
4. AIC/BIC准则支持模型参数选择

---

## 1. ARIMA模型概述

### 1.1 什么是ARIMA

ARIMA（AutoRegressive Integrated Moving Average）模型，即自回归积分滑动平均模型，是时间序列分析中最常用的预测模型之一。它由三个部分组成：

- **AR** (AutoRegressive)：自回归部分
- **I** (Integrated)：差分部分
- **MA** (Moving Average)：移动平均部分

### 1.2 模型表示

ARIMA模型通常表示为：**ARIMA(p, d, q)**

| 参数 | 含义 | 说明 |
|:----:|:-----|:-----|
| **p** | 自回归阶数 | 使用过去p个时刻的值进行回归 |
| **d** | 差分阶数 | 对原始序列进行d阶差分使其平稳 |
| **q** | 移动平均阶数 | 使用过去q个时刻的预测误差进行修正 |

## 2. ARIMA数学原理

### 2.1 自回归模型 AR(p)

自回归模型使用变量自身的历史值来回归解释当前值。

**公式：**

$$X_t = c + \sum_{i=1}^{p} \phi_i X_{t-i} + \varepsilon_t$$

其中：
- $X_t$：时刻t的时间序列值
- $c$：常数项
- $\phi_i$：自回归系数
- $p$：自回归阶数
- $\varepsilon_t$：白噪声误差项

**AR(1)模型示例：**

$$X_t = c + \phi_1 X_{t-1} + \varepsilon_t$$

### 2.2 差分运算 I(d)

差分运算用于将非平稳序列转化为平稳序列。

**一阶差分：**

$$\nabla X_t = X_t - X_{t-1}$$

**d阶差分：**

$$\nabla^d X_t = (1 - B)^d X_t$$

其中$B$为滞后算子：$B X_t = X_{t-1}$

### 2.3 移动平均模型 MA(q)

移动平均模型使用历史预测误差来修正当前值。

**公式：**

$$X_t = \mu + \varepsilon_t + \sum_{i=1}^{q} \theta_i \varepsilon_{t-i}$$

其中：
- $\mu$：序列均值
- $\theta_i$：移动平均系数
- $q$：移动平均阶数
- $\varepsilon_t$：白噪声误差项

**MA(1)模型示例：**

$$X_t = \mu + \varepsilon_t + \theta_1 \varepsilon_{t-1}$$

### 2.4 完整ARIMA模型

将AR、I、MA三部分结合，得到完整的ARIMA(p,d,q)模型：

$$(1 - \sum_{i=1}^{p} \phi_i B^i)(1 - B)^d X_t = c + (1 + \sum_{i=1}^{q} \theta_i B^i)\varepsilon_t$$

**展开形式：**

$$X_t' = c + \sum_{i=1}^{p} \phi_i X_{t-i}' + \varepsilon_t + \sum_{i=1}^{q} \theta_i \varepsilon_{t-i}$$

其中 $X_t' = \nabla^d X_t$ 是d阶差分后的序列。

## 3. ACF与PACF分析

### 3.1 自相关函数 ACF

**定义：**

ACF（Autocorrelation Function）衡量时间序列与其滞后版本之间的相关性。

$$\rho_k = \frac{\gamma_k}{\gamma_0} = \frac{Cov(X_t, X_{t-k})}{Var(X_t)}$$

其中：
- $\rho_k$：滞后k阶的自相关系数
- $\gamma_k$：滞后k阶的自协方差
- $\gamma_0$：方差

**ACF在ARIMA中的作用：**

| 模型类型 | ACF特征 |
|:--------|:--------|
| AR(p) | 拖尾（逐渐衰减） |
| MA(q) | 在q阶后截尾 |
| ARMA(p,q) | 拖尾 |

### 3.2 偏自相关函数 PACF

**定义：**

PACF（Partial Autocorrelation Function）衡量在控制中间滞后项影响后，时间序列与其滞后版本之间的直接相关性。

$$\phi_{kk} = Corr(X_t, X_{t-k} | X_{t-1}, X_{t-2}, ..., X_{t-k+1})$$

**PACF在ARIMA中的作用：**

| 模型类型 | PACF特征 |
|:--------|:--------|
| AR(p) | 在p阶后截尾 |
| MA(q) | 拖尾（逐渐衰减） |
| ARMA(p,q) | 拖尾 |

## 4. AIC与BIC准则

### 4.1 AIC准则（赤池信息准则）

**公式：**

$$AIC = 2k - 2\ln(L)$$

其中：
- $k$：模型参数个数
- $L$：似然函数值

**解释：**
- AIC值越小，模型越好
- 第一项$2k$惩罚复杂模型
- 第二项$-2\ln(L)$衡量模型拟合优度

### 4.2 BIC准则（贝叶斯信息准则）

**公式：**

$$BIC = k\ln(n) - 2\ln(L)$$

其中：
- $k$：模型参数个数
- $n$：样本数量
- $L$：似然函数值

**解释：**
- BIC值越小，模型越好
- 对复杂模型的惩罚比AIC更重
- 大样本时更倾向于选择简单模型

### 4.3 AIC与BIC比较

| 特性 | AIC | BIC |
|:-----|:----|:----|
| 惩罚项 | $2k$ | $k\ln(n)$ |
| 样本影响 | 较小 | 较大 |
| 模型选择 | 可能选择复杂模型 | 倾向于简单模型 |
| 一致性 | 不一致 | 一致 |
| 适用场景 | 预测为主 | 解释为主 |

## 5. 残差诊断

### 5.1 残差白噪声检验

**Ljung-Box检验：**

原假设 $H_0$：残差是白噪声（无自相关）

检验统计量：

$$Q = n(n+2)\sum_{k=1}^{h}\frac{\hat{\rho}_k^2}{n-k}$$

其中：
- $n$：样本数量
- $h$：滞后阶数
- $\hat{\rho}_k$：残差的k阶自相关系数

**判断标准：**
- p值 > 0.05：接受原假设，残差为白噪声
- p值 ≤ 0.05：拒绝原假设，残差存在自相关

### 5.2 Durbin-Watson检验

**统计量：**

$$DW = \frac{\sum_{t=2}^{n}(e_t - e_{t-1})^2}{\sum_{t=1}^{n}e_t^2}$$

**判断标准：**

| DW值 | 含义 |
|:-----|:-----|
| 接近0 | 存在正自相关 |
| 接近2 | 无自相关 |
| 接近4 | 存在负自相关 |

└─────────────────────────────────────────────────────────────────┘

步骤1: 数据准备

    ├── 处理缺失值和异常值
    └── 数据可视化观察趋势
    │
    ▼

    │
    ├── ADF检验 (Augmented Dickey-Fuller)
    ├── KPSS检验

    │
    ├── 如果平稳 ──► 步骤3

    │
    ├── 观察ACF和PACF图

    ▼

    │
    ├── 最大似然估计 (MLE)

    ├── 残差异方差检验

## 7. 航空客流预测中的ARIMA应用

### 7.1 数据特点

航空客流数据具有以下特点：

1. **季节性**：夏季和节假日客流高峰
2. **趋势性**：长期增长或下降趋势
3. **周期性**：周内模式（周末vs工作日）
4. **非平稳性**：通常需要差分处理

### 7.2 参数选择建议

基于航空客流数据的特点，建议参数范围：

| 参数 | 建议范围 | 理由 |
|:----:|:--------|:-----|
| p | 1-4 | 捕捉短期自相关 |
| d | 0-2 | 通常1阶差分即可平稳 |
| q | 0-3 | 捕捉随机冲击影响 |

### 7.3 模型优化策略

1. **网格搜索**：遍历参数组合寻找最优解
2. **交叉验证**：使用滚动预测评估模型性能
3. **残差诊断**：确保残差为白噪声
4. **综合评分**：结合预测精度和模型复杂度

## 8. 总结

ARIMA模型是时间序列预测的经典方法，其核心要点：

1. **平稳性**：通过差分使序列平稳
2. **定阶**：通过ACF/PACF或信息准则确定p,d,q
3. **诊断**：残差检验确保模型充分
4. **预测**：提供点预测和区间预测

在ATBS项目中，我们采用网格搜索+交叉验证的方法自动寻找最优ARIMA参数，确保预测精度和模型稳定性。
